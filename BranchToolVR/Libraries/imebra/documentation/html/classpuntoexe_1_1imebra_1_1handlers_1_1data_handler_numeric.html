Detailed Description

template<class dataHandlerType>
class puntoexe::imebra::handlers::dataHandlerNumeric< dataHandlerType >

This data handler accesses to the numeric data stored in a puntoexe::imebra::buffer class.

A special definition of this class (puntoexe::imebra::handlers::imageHandler) is used to access to the images' raw pixels.

#include <dataHandlerNumeric.h>
