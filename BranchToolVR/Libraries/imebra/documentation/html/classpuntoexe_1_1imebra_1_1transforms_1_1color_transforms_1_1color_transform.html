puntoexe::imebra::transforms::colorTransforms::colorTransform Class Referenceabstract
Transforms

Detailed Description

This is the base class used by the color transforms classes.

Each color transform class can handle a color space conversion from one color space to another.

The class colorTransformsFactory can select the right transform automatically when queried with the initial and the final color spaces as parameters.

#include <colorTransform.h>
