puntoexe::imebra::lut Class Reference
Image data

Detailed Description

Represents a Lookup table (LUT).

The lookup table maps a value stored into an image into another value that must be used for the visualization or for the analysis of the image.

3 lookups tables can be joined together to form a color palette.

#include <LUT.h>
