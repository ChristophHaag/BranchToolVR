Detailed Description

This class takes an image as an input and returns an 8 bit RGB bitmap of the requested image's area.

Renders the image specified in the constructor into an RGB or BGR buffer.

The caller can pass to the function a memory object that will be used to store the RGB/BGR buffer, otherwise the function will allocate a new one.

Each row of pixels in the RGB/BGR buffer will be aligned according to the template parameter rowAlignBytes

drawbitmap

The figure illustrates how the getBitmap() method works:

  1. the image is resized according to the parameters totalWidthPixels and totalHeightPixels
  2. the area specified by visibleTopLeftX, visibleTopLeftY - visibleBottomRightX, visibleBottomRightY is rendered into the buffer

Please note that the rendering algorithm achieves the described results without actually resizing the image.
