puntoexe::lockMultipleObjects Class Reference
Base classes

Detailed Description

Locks the access to several objects of type baseObject.

The class lock ALL the objects listed in the constructor's parameter. If the objects cannot be locked at once, then all of them are left unlocked until all of them can be locked all togheter.
