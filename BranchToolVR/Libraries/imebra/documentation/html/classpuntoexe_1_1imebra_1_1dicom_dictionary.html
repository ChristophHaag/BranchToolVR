puntoexe::imebra::dicomDictionary Class Reference
Dicom dictionary

Detailed Description

The Dicom Dictionary.

This class can be used to retrieve the tags' default data types and descriptions.

An instance of this class is automatically allocated by the library: your application should use the static function getDicomDictionary() in order to get the only valid instance of this class.

#include <dicomDict.h>
